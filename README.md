# Sales Performance Dashboard
## 📊 Project Overview
This project presents a Sales Performance Dashboard created using Excel and Power BI to analyze sales data and generate meaningful business insights.
## 🛠 Tools Used
- Microsoft Excel
- Power BI
## 📁 Files Included
- Dataset.xlsx → Raw dataset used for analysis
- Power BI dashboard.pbix → Power BI project file
## 📊 Dashboard Preview
![Dashboard](Screenshot%202026-03-26%20225210.png)
## 📈 Key Insights
- Analyzed sales and profit across different regions
- Compared product-wise performance to identify top-performing products
- Visualized sales trends over time for better decision-making
## 🚀 Outcome
Developed an interactive dashboard that helps track business performance and supports data-driven decision-making.
